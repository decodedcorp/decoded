## v2.0 Design System

### Import Path
```typescript
import {
  // Typography
  Heading, Text,
  // Inputs
  Input, SearchInput,
  // Cards
  Card, CardHeader, CardContent, CardFooter,
  ProductCard, GridCard, FeedCardBase, ProfileHeaderCard,
  // Headers & Footer
  DesktopHeader, MobileHeader, DesktopFooter,
  // Tokens
  typography, colors, spacing, shadows, borderRadius, zIndex
} from "@/lib/design-system"
```

### 컴포넌트 사용 가이드

| 컴포넌트 | 용도 | 예시 |
|----------|------|------|
| Heading | 제목 텍스트 | `<Heading variant="h2">제목</Heading>` |
| Text | 본문 텍스트 | `<Text variant="small">내용</Text>` |
| Card | 컨테이너 | `<Card variant="elevated" size="md">...</Card>` |
| ProductCard | 상품 표시 | `<ProductCard image={...} title="..." price="..."/>` |
| Input | 폼 입력 | `<Input variant="search" leftIcon={<Search/>}/>` |

### 디자인 토큰 참조

- **Typography**: `typography.sizes.h1`, `responsiveTypography.pageTitle`
- **Colors**: CSS variables via `colors.primary`, `colors.muted`
- **Spacing**: 4px base, `spacing[4]` = 16px
- **문서**: [docs/design-system/](docs/design-system/)
