1. **레이아웃 패턴**:
   - 반응형 그리드: 1/2/3/4 컬럼 패턴 (grid-cols-1 md:grid-cols-2 lg:grid-cols-3)
   - 섹션 래퍼: py-10 md:py-16, px-4 md:px-6 lg:px-8, max-w-7xl mx-auto
   - 교대 배경: card/background 패턴
   - Header padding: Desktop pt-16 (64px), Mobile pt-14 pb-14 (56px)

2. **애니메이션 패턴**:
   - AnimatePresence mode="wait" (탭/필터 전환)
   - Motion layoutId (탭 underline)
   - GSAP Flip (카드 레이아웃 전환)
   - Lenis smooth scroll
   - Direction-aware transitions (request flow: right-to-left forward, left-to-right backward)

3. **상태 관리 패턴**:
   - Loading state: isLoading prop + Loader2 spinner
   - Error state: error prop overrides variant
   - Empty state: Icon circle + message + CTA
   - Skeleton: CardSkeleton, ProductCardSkeleton 등

4. **컴포지션 패턴**:
   - Card slot components (CardHeader, CardContent, CardFooter)
   - asChild prop (Radix pattern)
   - Interactive cards: interactive prop + cursor-pointer + hover:shadow-lg
   - Wrapper strategy: Link when link prop, div when onClick prop

5. **테마 패턴**:
   - CSS variables for light/dark mode
   - next-themes ThemeProvider
   - 색상 사용 가이드 (primary: 주요 액션, secondary: 보조, destructive: 위험)
