Create NavItem component following decoded.pen Mobile Nav Bar spec and CVA pattern from tag.tsx:

**CVA Variants:**
- `state`: active (text-primary), inactive (text-muted-foreground)
- `size`: sm (icon 18px, text 9px), md (icon 22px, text 10px - default), lg (icon 26px, text 11px)

**Props Interface:**
- `icon`: React.ReactNode (required)
- `label`: string (required)
- `active`: boolean (for active state styling)
- `href`: string (optional, for Link navigation)
- `onClick`: () => void (optional, for action buttons like Request)
- `disabled`: boolean (optional)

**Styling per decoded.pen:**
- Layout: flex-col items-center gap-1
- Active: text-primary with 150ms transition
- Inactive: text-muted-foreground
- Hover (desktop only): text-foreground - use `@media (hover: hover)` to avoid sticky hover on touch
- Disabled: opacity-40, cursor-not-allowed

**Behavior:**
- Render as Link when `href` provided
- Render as button when `onClick` provided (for modal triggers like Request)
- Include aria-current="page" when active and using Link
- Include aria-label for accessibility
