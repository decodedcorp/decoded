  <files>packages/web/lib/design-system/hotspot.tsx</files>
  <action>
Create Hotspot component for marking items on images with clickable interactive markers.

Design specs from decoded.pen:
- Hotspot/Default: width 24px, height 24px, fill primary (#CFFF4C or var(--primary)), cornerRadius 12 (circle), pulsing animation
- Hotspot/Numbered: width 32px, height 32px, fill primary, cornerRadius 16, shows number text (Inter 14px fontWeight 600)
- Hotspot/Inactive: width 24px, height 24px, fill primary at 50% opacity (#CFFF4C80), cornerRadius 12, no animation

Implementation:
1. Use cva with 'variant' for default/numbered/inactive
2. Accept 'number' prop for numbered variant (optional, 1-99)
3. Accept onClick handler for interaction
4. Accept 'position' prop: { x: number, y: number } for absolute positioning (percentages)
5. Base styles: absolute rounded-full cursor-pointer transition-transform
6. Add pulsing animation for default variant using Tailwind's animate-pulse or custom keyframes
7. Custom animation: scale pulse effect (1 -> 1.2 -> 1) with opacity fade
8. Add hover state: scale-110
9. Add active state: scale-95
10. Use forwardRef pattern for button element

Variant styles:
- default: w-6 h-6 bg-primary animate-pulse-soft
- numbered: w-8 h-8 bg-primary text-primary-foreground font-semibold text-sm
- inactive: w-6 h-6 bg-primary/50 (no animation)

Add custom CSS animation in globals.css or use Tailwind's built-in animations.
Define pulse-soft animation: keyframes that scale and fade subtly.
