# Phase v2.1-01 Plan 02: StepIndicator & Hotspot Components Summary

StepIndicator for multi-step flow progress visualization and Hotspot for interactive item markers on images with pulsing animation.

## What Was Done

### Task 1: StepIndicator Component (569cf34)
Created `step-indicator.tsx` for visualizing progress in multi-step flows:

**Size Variants:**
- sm: 16px circles, 6px gap
- md: 24px circles, 8px gap (default)
- lg: 32px circles, 12px gap

**Visual Variants:**
- dots: Simple filled/unfilled circles (default)
- numbered: Circles with step number
- labeled: Circles with text labels below

**Features:**
- `currentStep` prop (1-indexed, required)
- `totalSteps` prop (default 3)
- Optional connector lines between steps
- Active steps show primary color, inactive show muted
- Accessibility: role="group", aria-label, aria-current

### Task 2: Hotspot Component (9960c38)
Created `hotspot.tsx` for interactive item markers on images:

**Variants:**
- default: 24x24px pulsing marker (animate-pulse-soft)
- numbered: 32x32px with digit display (1-99)
- inactive: 24x24px at 50% opacity, no animation

**Features:**
- `position` prop: `{ x: number, y: number }` as percentages
- Centered on position point with transform translate(-50%, -50%)
- Hover state: scale-110
- Active state: scale-95
- Focus-visible ring for accessibility
- `label` prop for accessible name

**Tailwind Config:**
Added custom `pulse-soft` keyframe animation that maintains centered positioning while scaling.

### Task 3: Barrel Exports (bd5ae4e)
Updated `index.ts` with exports:
- `StepIndicator`, `stepIndicatorVariants`, `StepIndicatorProps`
- `Hotspot`, `hotspotVariants`, `HotspotProps`, `HotspotPosition`

## Code Examples

### StepIndicator Usage
```tsx
import { StepIndicator } from "@/lib/design-system";

// Basic 3-step indicator on step 2
<StepIndicator currentStep={2} totalSteps={3} />

// Numbered variant with connectors
<StepIndicator currentStep={1} variant="numbered" showConnectors />

// Large labeled variant for upload flow
<StepIndicator
  currentStep={2}
  variant="labeled"
  labels={["Upload", "Detect", "Confirm"]}
  size="lg"
/>
```

### Hotspot Usage
```tsx
import { Hotspot } from "@/lib/design-system";

// Parent must have position: relative
<div className="relative">
  <Image src={imageUrl} alt="Fashion look" />

  {/* Default pulsing hotspot */}
  <Hotspot
    position={{ x: 35, y: 60 }}
    onClick={() => handleItemClick('jacket')}
    label="Blue denim jacket"
  />

  {/* Numbered hotspot */}
  <Hotspot
    variant="numbered"
    number={1}
    position={{ x: 20, y: 40 }}
    onClick={handleClick}
    label="Item 1: Sneakers"
  />

  {/* Already selected/viewed item */}
  <Hotspot
    variant="inactive"
    position={{ x: 50, y: 80 }}
    label="Selected item"
  />
</div>
```
