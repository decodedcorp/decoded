  <action>
Add TagGroup component to manage tag selection:

1. TagGroupProps interface:
   ```tsx
   interface TagGroupProps<T extends string = string> {
     items: { value: T; label: string }[];
     value: T | T[];  // single value or array for multi-select
     onChange: (value: T | T[]) => void;
     mode?: 'single' | 'multi';  // defaults to 'single'
     className?: string;
     tagSize?: 'sm' | 'default' | 'lg';
     tagVariant?: 'default' | 'outline';
   }
   ```

2. TagGroup implementation:
   - Renders a flex container with gap-2
   - Maps items to Tag components
   - Single mode: clicking tag sets value to that tag
   - Multi mode: clicking tag toggles it in/out of array
   - Pass isActive based on current selection

3. Example usage:
   ```tsx
   // Single select (category filter)
   <TagGroup
     items={[
       { value: 'all', label: 'All' },
       { value: 'latest', label: 'Latest' },
       { value: 'clothing', label: 'Clothing' },
     ]}
     value={selectedCategory}
     onChange={setSelectedCategory}
     mode="single"
   />

   // Multi select (tag selection)
   <TagGroup
     items={tags}
     value={selectedTags}
     onChange={setSelectedTags}
     mode="multi"
   />
   ```

4. Export TagGroup and TagGroupProps
