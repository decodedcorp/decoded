# Milestone v2.1: Design System Expansion

**Status:** SHIPPED 2026-02-06
**Phases:** v2.1-01 to v2.1-06
**Total Plans:** 14

## Overview

Expand decoded.pen design system with 23 new components grouped by function: interactive elements, navigation, cards, profile/detail, login/state, and visual QA automation.

## Milestone Summary

**Components Created:** 23
- Core Interactive: Tag, ActionButton, StepIndicator, Hotspot
- Navigation: NavBar, NavItem, SectionHeader, Tabs
- Cards: ArtistCard, StatCard, SpotCard, ShopCarouselCard
- Profile/Detail: Badge, LeaderItem, RankingItem, SpotDetail
- Login/State: OAuthButton, GuestButton, Divider, LoginCard, LoadingSpinner, SkeletonCard, BottomSheet

**Key Decisions:**
- CVA pattern for all component variants (consistent with existing)
- forwardRef pattern for component refs
- Custom keyframes in tailwind config for animations
- React Context for Tabs state sharing
- Inline SVG icons for OAuth providers

**Issues Resolved:**
- Design system gap: decoded.pen components now implemented
- Visual QA: Automated testing infrastructure established

**Tech Debt Incurred:**
- 15 orphaned components (exported but not integrated)
- Duplicate implementations exist for some components (LoginCard, StepIndicator)
- Integration score: 35% (8/23 components actively used)
