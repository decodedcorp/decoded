    **1. Create `packages/web/lib/components/magazine/types.ts`:**

    Define types matching the FLW-06 Layout JSON Contract exactly:

    ```typescript
    // Component types from FLW-06 registry
    type ComponentType = 'hero-image' | 'text-block' | 'item-card' | 'divider' | 'quote' | 'grid-gallery';
    type AnimationType = 'fade-up' | 'scale-in' | 'slide-left' | 'parallax' | 'none';

    interface LayoutComponent {
      type: ComponentType;
      x: number;  // percentage 0-100
      y: number;  // percentage 0-100
      w: number;  // percentage width
      h: number;  // percentage height
      animation_type: AnimationType;
      animation_delay?: number;
      data: Record<string, unknown>;
    }

    interface LayoutJSON {
      version: 1;
      viewport: 'mobile' | 'desktop';
      components: LayoutComponent[];
    }

    interface ThemePalette {
      primary: string;    // e.g. "#050505"
      accent: string;     // e.g. "#eafd67"
      bg: string;         // background
      text: string;       // text color
    }

    interface MagazineIssue {
      id: string;
      issue_number: number;
      title: string;
      subtitle?: string;
      cover_image_url: string;
      layout_json: LayoutJSON;
      theme_palette: ThemePalette;
      generated_at: string;        // ISO date
      theme_keywords?: string[];   // For collection grouping
    }

    // Personal issue status from SCR-MAG-02
    type PersonalStatus = 'idle' | 'checking' | 'generating' | 'ready' | 'error';
    ```

    Export all types.

    **2. Create `packages/web/lib/components/magazine/theme.ts`:**

    - Define `defaultMagazineTheme: ThemePalette` with Deep Black (#050505) / Neon Chartreuse (#eafd67):
      ```
      { primary: "#050505", accent: "#eafd67", bg: "#050505", text: "#f5f5f5" }
      ```
    - Create `injectMagazineTheme(palette: ThemePalette, container: HTMLElement)` function that sets CSS custom properties `--mag-primary`, `--mag-accent`, `--mag-bg`, `--mag-text` on the container element.
    - Create `removeMagazineTheme(container: HTMLElement)` to clean up.

    **3. Add magazine CSS custom properties to `globals.css`:**

    Add a section (after existing custom properties) with default magazine theme values:
    ```css

    :root {
      --mag-primary: #050505;
      --mag-accent: #eafd67;
      --mag-bg: #050505;
      --mag-text: #f5f5f5;
    }
    ```

    **4. Extend `tailwind.config.ts`:**

    Add magazine theme colors under `extend.colors`:
    ```
    mag: {
      primary: 'var(--mag-primary)',
      accent: 'var(--mag-accent)',

    }
    ```
    This enables classes like `bg-mag-bg`, `text-mag-accent`, `border-mag-accent`.

    Do NOT modify existing color definitions — only add the `mag` namespace.
