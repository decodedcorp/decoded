## v3.0 Requirements

### Admin Authentication & Layout

- [x] **AAUTH-01**: 미들웨어 레벨의 관리자 인증 — is_admin 체크로 /admin 하위 전체 경로 보호
- [x] **AAUTH-02**: 라우트 보호 + 레이아웃 더블 체크 — 비관리자 즉시 리다이렉트
- [x] **AAUTH-03**: 다크 사이드바 레이아웃 — 5개 네비게이션 링크 (Dashboard, AI Audit, AI Cost, Pipeline Logs, Server Logs)

### Dashboard

- [x] **DASH-01**: 5개 KPI 카드 (DAU, MAU, 총 유저, 총 포스트, 총 솔루션) — 실데이터+mock 하이브리드
- [x] **DASH-02**: Recharts AreaChart — 기간 선택기(7D/14D/30D)로 DAU/검색/클릭 추이 시각화
- [x] **DASH-03**: 오늘의 요약 — 당일 신규 포스트, 솔루션, 클릭 수 표시

### AI Audit

- [x] **AUDIT-01**: 감사 테이블 — 썸네일, 상태, 아이템 수 표시 + 페이지네이션
- [x] **AUDIT-02**: 상세 모달 — 핫스팟 오버레이 + 신뢰도 점수 + 양방향 하이라이트
- [x] **AUDIT-03**: 인라인 편집 — 아이템 추가/삭제/수정 (클라이언트 사이드, mock 범위)
- [x] **AUDIT-04**: 상태 필터 — 5개 버튼 (All/Pending/Completed/Error/Modified)

### AI Cost Monitoring

- [x] **COST-01**: API 호출 BarChart — 기간 선택기(7D/30D/90D)로 일별 API 호출 수 시각화
- [x] **COST-02**: 토큰 사용량 stacked AreaChart — 입력/출력 토큰 분리 시각화
- [x] **COST-03**: 모델 비용 테이블 + KPI 카드 — 전기 대비 델타 표시

### Pipeline Logs

- [x] **PIPE-01**: 파이프라인 목록 — 상태/단계/소요시간 컬럼 + 필터 + 페이지네이션
- [x] **PIPE-02**: 아코디언 상세 — 클릭 시 단계별 타임라인 (upload→analyze→detect) 렌더링 — v3-06에서 응답 형태 수정
- [x] **PIPE-03**: 재시도 버튼 — 실패 파이프라인에서 재시도 동작 (sonner toast) — v3-06에서 접근성 복원

### Server Logs

- [x] **SLOG-01**: 로그 테이블 — 엔드포인트, HTTP 상태코드, 응답시간 표시
- [x] **SLOG-02**: 필터 — 레벨/시간/검색어 필터 + URL 동기화
- [x] **SLOG-03**: 터미널 스트리밍 — tail -f 스타일 실시간 로그 표시 (2.5s polling, 자동 스크롤)
