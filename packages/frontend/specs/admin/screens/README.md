## 권한 체계

| 역할 | 대시보드 | 콘텐츠 관리 | 지불 관리 |
|:---|:---:|:---:|:---:|
| Super Admin | ✅ | ✅ | ✅ |
| Content Admin | ✅ | ✅ | ❌ |
| Finance Admin | ✅ | ❌ | ✅ |
| Viewer | ✅ (읽기 전용) | ❌ | ❌ |

## 대시보드 위젯

### 통계 카드
- 총 사용자 수 / 신규 가입자
- 총 이미지 수 / 오늘 업로드
- 대기 중 신고 건수
- 미정산 금액

### 차트
- 일별 가입자 추이
- 카테고리별 이미지 분포
- 지역별 트래픽

### 최근 활동
- 최근 신고 목록
- 최근 업로드 목록
- 최근 정산 요청

## 보안 고려사항

- 관리자 경로는 별도 인증 미들웨어 적용
- 모든 관리 작업에 감사 로그 기록
- IP 화이트리스트 적용 가능
- 세션 타임아웃 30분 (일반 사용자 대비 짧게)
