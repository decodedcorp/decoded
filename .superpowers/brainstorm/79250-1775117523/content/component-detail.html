Single strategy: actual ratio + clamp. Zero crop everywhere.

Core: useImageDimensions Hook

Check localStorage → fast // 3. Create Image() → onload → save to both // Returns: { width, height, ratio, loading } } // Clamp: min 0.5 (2:1 landscape) ~ max 2.0 (1:2 portrait) function clampRatio(ratio: number): number { return Math.min(2.0, Math.max(0.5, ratio)) } // CSS-ready output function getAspectStyle(w, h, fallback = "3/4") { if (!w || !h) return { aspectRatio: fallback } const r = clampRatio(h / w) return { aspectRatio: `${1} / ${r}` } }

Edge Cases & Clamping

1:2 (too tall)
→ clamped to 1:2
3:4 (normal)
→ as-is
1:1 (normal)
→ as-is
2:1 (too wide)
→ clamped to 2:1
Clamp range: 0.5 (2:1 landscape) ~ 2.0 (1:2 portrait)
Dimension 미확보 시: 기존 fallback 비율 유지 (FeedCard→4:5, Explore→3:4)
Transition: fallback → real ratio, opacity+scale micro-animation (0.15s)
