# Phase v2.1-05 Plan 02: LoginCard & LoadingSpinner Summary

**One-liner:** LoginCard with glassmorphism effect composing OAuth/guest buttons, plus pill-shaped LoadingSpinner for async states.

## What Was Built

### LoadingSpinner Component
- Pill-shaped loading indicator with spinning Loader2 icon
- Dark background with opacity (`bg-neutral-900/80`)
- Configurable text prop (default: "Loading...")
- Location: `packages/web/lib/design-system/loading-spinner.tsx`

### LoginCard Component
- Glassmorphism effect: `bg-white/5 backdrop-blur-xl rounded-2xl`
- "decoded" logo in primary color (#d9fc69) with Playfair Display font
- Composes three design-system components:
  - OAuthButton (Kakao, Google, Apple)
  - Divider
  - GuestButton
- Error message display with red background
- Terms of Service and Privacy Policy footer links
- Full loading state management via props
- Location: `packages/web/lib/design-system/login-card.tsx`

### Barrel Exports
- Both components exported from `@/lib/design-system`
- Types exported: `LoginCardProps`, `LoadingSpinnerProps`
