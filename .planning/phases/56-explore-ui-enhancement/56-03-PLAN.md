**MagazineEditorialSection.tsx:**

1. The pull quote accent line decorations (lines 97-100, 107-110) currently use:
```tsx
backgroundColor: accentColor
  ? `${accentColor}4D`
  : "var(--primary-30, hsl(var(--primary) / 0.3))"
```
Since `accentColor` will now be `"var(--mag-accent)"` (a CSS variable string, not a hex), appending `4D` (hex alpha) won't work. Change to:
```tsx
backgroundColor: "color-mix(in srgb, var(--mag-accent) 30%, transparent)"
```
OR if color-mix is a concern, use a simpler approach:
```tsx
backgroundColor: "var(--mag-accent)",
opacity: 0.3,
```
But opacity affects the whole element. Best approach: wrap the accent line `div` with the opacity approach, or use a dedicated CSS class. Simplest working solution:
```tsx
style={{
  backgroundColor: "var(--mag-accent)",
  opacity: 0.3,
}}
```

2. The `accentStyle` block (lines 63-65) that sets `--magazine-accent` from the prop can be simplified. Since the parent always passes `"var(--mag-accent)"`, this still works correctly. But for robustness, change the fallback:
