  <action>
Create DesktopFooter component following design-system patterns:

1. **Overall Structure:**
   - Full-width footer with bg-muted or bg-card background
   - Two sections: Main content grid + Bottom bar
   - Padding: py-12 px-6 for main, py-4 px-6 for bottom
   - role="contentinfo" for accessibility

2. **Main Content - Desktop (md+):**
   - 4-column grid: `grid-cols-4 gap-8`
   - Column 1 (Brand): Logo + slogan text
   - Column 2 (Company): About, Careers, Press, Blog links
   - Column 3 (Support): Help, Contact, FAQ, Report links
   - Column 4 (Connect): Social icons + Newsletter form

3. **Main Content - Mobile (<md):**
   - Stack layout: `flex flex-col gap-6`
   - Brand section always visible
   - Company and Support as collapsible accordions (use useState)
   - Connect section always visible (no accordion)

4. **Brand Section (FooterBrand):**
   - DecodedLogo component (small variant)
   - Slogan text: "Discover fashion from your favorite celebrities"
   - Use Text component from design-system

5. **Link Sections (FooterLinkSection):**
   - Reusable sub-component with:
     - title: string
     - links: Array<{ label: string; href: string }>
     - collapsible?: boolean (for mobile)
   - Title styled with font-medium, mb-4
   - Links as vertical list with text-muted-foreground, hover:text-foreground
   - If collapsible, wrap content in accordion pattern with ChevronDown icon

6. **Social Links (FooterSocialLinks):**
   - Horizontal row of icon buttons: Instagram, Twitter, Facebook
   - Use lucide-react icons
   - External links with target="_blank" rel="noopener noreferrer"
   - Icon size: h-5 w-5

7. **Newsletter (FooterNewsletter):**
   - Label text: "Newsletter"
   - Input field (use Input from design-system) with placeholder "Email..."
   - Submit button (use Button with variant="ghost" and ArrowRight icon)
   - Flex row layout
   - Optional: form submission handler prop onSubscribe?: (email: string) => void

8. **Bottom Bar (FooterBottom):**
   - Left: Copyright text "© 2026 DECODED. All rights reserved."
   - Center: Legal links (Privacy Policy, Terms of Service, Cookie Policy)
   - Right: Language selector (simple select or dropdown)
   - On mobile: stack vertically
   - Separator line above (border-t border-border)

9. **Props Interface:**
```typescript
interface DesktopFooterProps {
  className?: string;
  onNewsletterSubscribe?: (email: string) => void;
}
```

10. **Link Data:**
Define static link arrays within the component:
```typescript
const companyLinks = [
  { label: 'About', href: '/about' },
  { label: 'Careers', href: '/careers' },
  { label: 'Press', href: '/press' },

const supportLinks = [
  { label: 'Help Center', href: '/help' },
  { label: 'Contact', href: '/contact' },
  { label: 'FAQ', href: '/faq' },
  { label: 'Report Issue', href: '/report' },
];

const socialLinks = [
  { icon: Instagram, href: 'https://instagram.com/decoded_app', label: 'Instagram' },
  { icon: Twitter, href: 'https://twitter.com/decoded_app', label: 'Twitter' },
  { icon: Facebook, href: 'https://facebook.com/decoded_app', label: 'Facebook' },
];
```
