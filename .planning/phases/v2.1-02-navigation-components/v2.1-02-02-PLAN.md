Create generic Tabs component following decoded.pen spec and SearchTabs pattern:

**Tabs Container Props (TabsProps):**
- `children`: React.ReactNode (TabItem children)
- `value`: string (current active tab id)
- `onValueChange`: (value: string) => void
- `className`: string (optional)
- `layoutId`: string (optional, default: "tabs-underline" - for motion layoutId)

**TabItem Props (TabItemProps):**
- `value`: string (unique tab id)
- `children`: React.ReactNode (tab label content)
- `className`: string (optional)
- `count`: number (optional, displays as "Label (count)")

**CVA Variants for TabItem:**
- `state`: active (text-foreground), inactive (text-muted-foreground hover:text-foreground)

**Styling per decoded.pen:**
- Tabs container: flex items-center gap-1 border-b border-border
- TabItem: relative px-4 py-3 text-sm font-medium transition-colors
- Active indicator: motion.span with h-0.5 bg-primary absolute bottom-0 left-0 right-0

**Sliding Underline Animation:**
- Use motion.span with layoutId prop (from Tabs container or default)
- Spring animation: { type: "spring", stiffness: 500, damping: 30 }
- Only render motion.span on active TabItem

**Count Display:**
- Format: "Label (count)" when count > 0
- Show "999+" when count > 999

**Accessibility:**
- Tabs container: role="tablist"
- TabItem: role="tab", aria-selected={isActive}

**Implementation Pattern:**
Use React Context to pass active value and onValueChange from Tabs to TabItem:
```tsx
const TabsContext = createContext<{
  value: string;
  onValueChange: (value: string) => void;
  layoutId: string;
} | null>(null);
```
