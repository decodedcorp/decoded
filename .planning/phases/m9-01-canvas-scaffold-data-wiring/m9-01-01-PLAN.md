**scatter.ts** ÔÇö Implement djb2 seeded PRNG scatter engine (pure functions, zero DOM dependency):

- IMPORTANT: Do NOT re-define ScatterPosition here. Import it from `./types` and re-export as convenience: `export type { ScatterPosition } from './types';`

1. `djb2(str: string): number` ÔÇö standard djb2 hash: `hash = 5381`, loop `hash = ((hash << 5) + hash) ^ charCode`, return `hash >>> 0`
2. `seededRandom(seed: string, salt: number): number` ÔÇö returns float in [0, 1) via `(djb2(seed + String(salt)) % 10000) / 10000`
3. `computeScatterPosition(id: string, index: number): ScatterPosition` ÔÇö zone-based scatter:
   - Grid: COLS=4, ROWS=3 (12 zones)
   - Zone assignment: `col = index % COLS`, `row = floor(index / COLS) % ROWS`
   - Zone size: `zoneW = 100/COLS` (25%), `zoneH = 100/ROWS` (~33.3%)
   - Random offset within zone: `xOffset = seededRandom(id, 1) * zoneW * 0.7`, `yOffset = seededRandom(id, 2) * zoneH * 0.7`
   - Position: `left = zoneCol * zoneW + xOffset`, `top = zoneRow * zoneH + yOffset`
   - Cap left at 82% max, top at 80% max (prevent edge clipping per RESEARCH.md pitfall 2)
   - Rotation: `(seededRandom(id, 3) - 0.5) * 24` ÔÇö range -12 to +12 degrees
   - z-index: `floor(seededRandom(id, 4) * 20) + 1` ÔÇö range 1-20
   - Size variation: `sizeVariant = 0.85 + seededRandom(id, 5) * 0.35` (0.85-1.2x), width = `clamp(${round(100*sv)}px, ${round(15*sv)}vw, ${round(220*sv)}px)`
   - CRITICAL: No Math.random() anywhere ÔÇö only djb2 + seededRandom

Export `computeScatterPosition`. Re-export `ScatterPosition` type from types.ts.

**index.ts** ÔÇö Barrel exports: re-export everything from `types.ts` and `scatter.ts`. Leave placeholder comment for future component exports (MainPageD, PolaroidCard will be added in Plan 02).
