# Phase v2.1-01 Plan 01: Tag & ActionButton Components Summary

Tag component with 6 category color variants and ActionButton with 3 interaction variants, following decoded.pen design specs.

## What Was Done

### Task 1: Tag Component (5b7c360)
Created `tag.tsx` with category-specific color variants matching decoded.pen:
- **all**: Dark (#1A1A2E) background, white text
- **latest**: Amber-100 background, amber-800 text
- **clothing**: Pink-100 background, pink-900 text
- **accessories**: Violet-100 background, violet-800 text
- **shoes**: Sky-100 background, sky-800 text
- **bags**: Stone-100 background, stone-600 text

Features:
- Size variants: sm (24px), md (32px), lg (40px)
- Active state with ring indicator
- Button element with forwardRef for accessibility

### Task 2: ActionButton Component (ebad7f5)
Created `action-button.tsx` with 3 variants:
- **default**: Semi-transparent black (bg-black/50), white icon - for overlays
- **solid**: Primary color background - for emphasis
- **outline**: Transparent with border - for standard UI

Features:
- Size variants: sm (32px), md (40px), lg (48px)
- Hover/active/disabled states
- Icon prop for flexible icon rendering

### Task 3: Barrel Exports (bc1447a)
Updated `index.ts` with exports:
- `Tag`, `tagVariants`, `TagProps`
- `ActionButton`, `actionButtonVariants`, `ActionButtonProps`

## Code Examples

### Tag Usage
```tsx
import { Tag } from "@/lib/design-system";

// Category filtering
<Tag category="clothing" onClick={() => filter('clothing')}>Clothing</Tag>
<Tag category="shoes" size="sm">Shoes</Tag>
<Tag category="all" active>All</Tag>
```

### ActionButton Usage
```tsx
import { ActionButton } from "@/lib/design-system";
import { Heart, Bookmark, Share } from "lucide-react";

// Overlay actions (on images)
<ActionButton icon={<Heart className="h-5 w-5" />} />

// Emphasized action
<ActionButton icon={<Bookmark className="h-5 w-5" />} variant="solid" />

// Standard UI action
<ActionButton icon={<Share className="h-5 w-5" />} variant="outline" size="sm" />
```
