<decisions>
## Implementation Decisions

### ArtistCard 정보 구성
- 팔로우 버튼 없음 — 카드 클릭 시 프로필 페이지로 이동만
- 표시 정보: 아바타 + 이름 + 포스트 수 (팔로워 수 대신)
- 크기: 반응형 (min 120px), 그리드 컨테이너에 따라 유동적으로 조정
- 클릭 동작: /profile/[id] 페이지로 네비게이션

### StatCard 수치 포맷
- 큰 숫자 포맷: 1.2K / 3.5M 형식으로 축약 (1000 → 1K, 1000000 → 1M)
- 증감 표시 포함: 화살표 + 증감 수치 (예: ↑ +15%)
- 통계 종류 예시: Views, Likes, Posts, Followers
- 레이아웃: 2×2 그리드로 4개 카드 고정 배치

### SpotCard 3 variants 전환
- Default→Active 전환 트리거: Hotspot 클릭과 SpotCard 자체 클릭 모두 지원
- Compact 사용 컨텍스트: 모바일 리스트 뷰 + 데스크톱 사이드 패널 모두
- 표시 정보: 상품 이미지 + 브랜드 + 이름 + 가격
- 클릭 동작: 앱 내 상품 상세 모달 오픈

### ShopCarouselCard 인터랙션
- 스와이프 방식: 카드 단위 스냅 (각 카드 위치에 정렬)
- 표시 정보: 이미지 + 브랜드 + 이름 + 가격
- 클릭 동작: 외부 쇼핑몰 구매 링크 열기 (새 탭)
- 페이지 인디케이터(dots): 표시 안함 — 깨끗한 캐러셀 유지

### Claude's Discretion
- SpotCard variant 간 트랜지션 애니메이션 (duration, easing)
- ShopCarouselCard 스냅 스크롤 구현 방식 (CSS snap vs JS)
- StatCard 증감 화살표 아이콘 스타일 (↑↓ vs Lucide icons)
- 카드 hover 상태 세부 스타일링

</decisions>

<specifics>
## Specific Ideas

- ShopCarouselCard는 외부 링크로 직접 연결 (앱 내 모달 불필요)
- SpotCard는 앱 내 상세 모달로 연결 (추가 정보 제공 후 구매 결정)
- ArtistCard는 팔로워 수 대신 포스트 수를 표시하여 콘텐츠 활동량 강조

</specifics>
