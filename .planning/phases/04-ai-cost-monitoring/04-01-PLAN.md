const AI_MODELS = [

**`deterministicInt(seed: string, modulo: number): number`**

Same djb2 hash function as `mock-data.ts`:
```typescript
function deterministicInt(seed: string, modulo: number): number {
  let hash = 5381;
  for (let i = 0; i < seed.length; i++) {
    hash = ((hash << 5) + hash) ^ seed.charCodeAt(i);
    hash = hash >>> 0;
  }
  return hash % modulo;
}
```

**`generateAiCostMetrics(days: number = 30): AiCostDailyMetric[]`**

Generate `days` days of daily AI cost data ending at today.

For each day (going backwards from today):
1. Compute date string `YYYY-MM-DD`
2. API calls: base 65, weekday range 50-120, weekend dip to 30-60. Use `deterministicInt(date + ":calls", 71) + 50` for weekdays, `deterministicInt(date + ":calls", 31) + 30` for weekends. Add slight upward trend: `+ Math.floor(dayIndex * 0.3)` where dayIndex 0 = oldest day.
3. Input tokens: `deterministicInt(date + ":input", 150001) + 50000` (range 50K-200K)
4. Output tokens: roughly 20-30% of input — `deterministicInt(date + ":output", 40001) + 10000` (range 10K-50K)
5. Estimated cost: compute from tokens using a blended rate. For simplicity, use weighted average of model pricing:
   - `inputCost = (inputTokens / 1000) * weightedInputPrice`
   - `outputCost = (outputTokens / 1000) * weightedOutputPrice`
   - Where weighted prices are: `sum(model.inputPricePerK * model.weight)` and same for output
   - Round to 4 decimal places for daily cost

Sort results chronologically (oldest first) for chart display.

**`generateAiCostKPI(days: number = 30): AiCostKPI`**

1. Generate metrics for `days` (current period)
2. Generate metrics for `days` (previous period — the `days` before that)
3. Sum totals for current period: totalCalls, totalTokens (input+output), totalCost
4. Compute avgCostPerCall = totalCost / totalCalls
5. Compute deltas as percentage: `((current - previous) / previous) * 100`, rounded to 1 decimal
6. Return KPI object

**`generateModelCostBreakdown(days: number = 30): ModelCostBreakdown[]`**

For each model in `AI_MODELS`:
1. Generate total calls for period: sum daily apiCalls from `generateAiCostMetrics(days)`, multiply by model `weight`
2. Distribute tokens proportionally: total input/output tokens * weight, but add slight variance using deterministicInt
3. Compute cost: `(inputTokens / 1000) * model.inputPricePerK + (outputTokens / 1000) * model.outputPricePerK`
4. Round costs to 2 decimal places

Return array of 3 model breakdowns.

**Create `packages/web/app/api/v1/admin/ai-cost/kpi/route.ts`** (GET):

Follow the standard admin auth pattern:

```typescript
import { NextRequest, NextResponse } from "next/server";
import { createSupabaseServerClient } from "@/lib/supabase/server";
import { checkIsAdmin } from "@/lib/supabase/admin";
import { fetchAiCostKPI } from "@/lib/api/admin/ai-cost";

export async function GET(request: NextRequest) {
  const supabase = await createSupabaseServerClient();
  const { data: { user } } = await supabase.auth.getUser();
  if (!user) {
    return NextResponse.json({ error: "Unauthorized" }, { status: 401 });
  }
  const isAdmin = await checkIsAdmin(supabase, user.id);
  if (!isAdmin) {
    return NextResponse.json({ error: "Forbidden" }, { status: 403 });
  }

  const { searchParams } = request.nextUrl;
  const days = Math.min(90, Math.max(7, parseInt(searchParams.get("days") ?? "30", 10)));

  const data = await fetchAiCostKPI(days);
  return NextResponse.json(data);
}
```
