Create RankingItem component for displaying rank changes.

**Component structure:**
- Use CVA for size variants: `sm` (inline in LeaderItem), `md` (standalone display), `lg` (profile page feature)
- Props: `direction` ('up' | 'down' | 'neutral'), `value` (number), `size` (sm/md/lg), `className`

**Visual format:**
- Up: TrendingUp icon + "+{value}" text, primary color (green: `text-green-600 dark:text-green-400`)
- Down: TrendingDown icon + "-{value}" text, destructive color (red: `text-red-600 dark:text-red-400`)
- Neutral: Minus icon + "0" or dash, muted color (`text-muted-foreground`)

**Size variants:**
- `sm`: Icon 14px, text text-xs, gap-0.5 (for embedding in LeaderItem)
- `md`: Icon 16px, text text-sm, gap-1 (default standalone)
- `lg`: Icon 20px, text text-base font-semibold, gap-1.5 (profile feature)

**Important:** Show actual numbers (not percentages). Format: "+5", "-12", "0"

Follow stat-card.tsx pattern for trend display, but make it standalone and reusable.

**Layout (image overlay style per CONTEXT.md):**

   - Full width image with aspect-[4/5] or aspect-[3/4]
   - Gradient overlay at bottom (transparent to black/dark)
   - Brand + name overlaid on bottom of image
   - Price range shown if multiple shops

2. **Info overlay on image:**
   - Position absolute at bottom
   - Text: Brand (small, muted), Name (large, bold), Price range
   - Background: gradient overlay for readability

   - List of shopping options
