# Phase v2.1-04: Profile & Detail Components - Context

**Gathered:** 2026-02-06
**Status:** Ready for planning

<domain>
## Phase Boundary

프로필 페이지의 게이미피케이션 요소(배지, 리더보드, 랭킹)와 스팟 상세 정보 컴포넌트를 decoded.pen 디자인 시스템으로 구현. 사용자가 획득한 배지, 순위, 상세 상품 정보를 일관된 스타일로 확인할 수 있음.

</domain>

<decisions>
## Implementation Decisions

### Badge 스타일링
- 고정 그리드 레이아웃 (3-4열)으로 전체 배지 표시
- 잠금된 배지(미획득): 어두운 실루엣 + 물음표(?) 표시로 미스터리 감
- 배지 등급/희귀도: 아이콘 디자인으로 구분 (테두리 색상이나 크기가 아닌 아이콘 자체로)
- 배지 클릭 시: 모달/바텀시트로 상세 정보 표시 (배지 이름, 획득 조건, 획득 일자 등)

### Leaderboard 표현
- 포디움(1-3위) 강조: 금/은/동 메달 아이콘 + 행 배경색으로 구분
- 각 항목 정보: 아바타 + 닉네임 + 점수 + 랭크 변동 표시
- 나 자신 표시: 하단 고정 영역 + 리스트 내 배경색 하이라이트 (둘 다 적용)
- 리더보드 아이템 클릭: 프로필 미리보기 모달로 간단 정보 확인

### Ranking 변동 UI
- 시각화: 화살표 + 숫자 형식 (↑+5, ↓-3)
- 색상: 브랜드 컬러 사용 (상승=primary, 하락=secondary 계열, 유지=neutral)
- 사용처: LeaderItem 내부 서브 컴포넌트 + 프로필 페이지 독립 사용 (둘 다 지원)
- 수치 포맷: 실제 숫자 그대로 표시 (+5, -12 등)

### SpotDetail 레이아웃
- 정보 배치: 이미지 위에 정보 오버레이 형태
- 구매 링크: 여러 쇼핑몰 링크를 리스트로 나열
- 가격 표시: 각 쇼핑몰별 개별 가격 표시
- 추가 정보: 하단에 비슷한 상품 추천 영역 포함

### Claude's Discretion
- 배지 그리드의 정확한 열 수 (모바일/데스크탑 반응형)
- 메달 아이콘 구체적인 디자인
- 오버레이 투명도 및 블러 강도
- 비슷한 상품 추천 카드 레이아웃 (기존 SpotCard 재사용 여부)

</decisions>
