    **Tilt Animation (GSAP):**
    - Track mouse position relative to container center
    - On `mousemove`: use `gsap.to()` to rotate the celebrity name element:
      - `rotateX`: map mouseY offset to -8deg to +8deg range
      - `rotateY`: map mouseX offset to -8deg to +8deg range
      - Use `perspective: 800px` on parent, `transformStyle: preserve-3d` on text
      - Duration: 0.4s for smooth follow (not instant)
    - On `mouseleave`: reset to `rotateX: 0, rotateY: 0`
    - On mobile (use `window.matchMedia('(hover: hover)')` check): skip tilt entirely, no event listeners
    - GSAP context for cleanup on unmount

    **Noise overlay:**
    - Use CSS background with SVG noise: `bg-[url('https://grainy-gradients.vercel.app/noise.svg')]`
    - `opacity-[0.04]`, `mix-blend-overlay`, `pointer-events-none`
    - Full absolute covering the hero section

    **Entry animation (GSAP timeline):**
    - Background image: scale 1.15 -> 1.0, opacity 0 -> 0.3, duration 2s
    - Label: opacity 0 -> 1, y 20 -> 0, delay 0.3s
    - Celebrity name: opacity 0 -> 1, y 40 -> 0, delay 0.5s
    - Title: opacity 0 -> 1, y 30 -> 0, delay 0.7s
    - CTA button: opacity 0 -> 1, y 20 -> 0, delay 0.9s
    - NeonGlow: opacity 0 -> 1, scale 0.8 -> 1, delay 0.2s

    - Use mag-* Tailwind classes from m7-01 theme: `bg-mag-bg`, `text-mag-text`

    - Font: celebrity name uses `font-serif` (Playfair Display already loaded), italic, bold
    - Responsive: min-h-[85vh] on mobile, min-h-screen on desktop

    **Important:** Import GSAP the same way as MagazineRenderer: `import gsap from 'gsap'`. Do NOT import motion from "motion/react" -- use GSAP exclusively for consistency with the magazine system.
