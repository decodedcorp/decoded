# Quick Task 260402-n75: 포스트 상세 페이지 크게보기 버튼 클릭시 페이지 전환 안됨

## Root Cause

`ImageDetailModal`의 "크게보기" 버튼 → `handleMaximize()` → `router.push('/posts/${imageId}')` 호출 시,
Next.js intercepting route `@modal/(.)posts/[id]`가 soft navigation을 다시 가로채서 모달만 재렌더링됨.
실제 `/posts/[id]` fullpage route로 이동하지 않음.

## Fix

`router.push()` → `window.location.href` 변경 (hard navigation)으로 intercepting route 우회.
GSAP 애니메이션은 유지 — fade-out 완료 후 hard navigation 실행.
