# Quick Task 260402-n75: Summary

## Task
포스트 상세 페이지 크게보기 버튼 클릭시 페이지 전환 안됨

## Root Cause
`handleMaximize()`에서 `router.push('/posts/${imageId}')`를 호출하면 Next.js intercepting route `@modal/(.)posts/[id]`가 soft navigation을 가로채서 모달만 재렌더링됨.

## Fix
`router.push()` → `window.location.href` (hard navigation)으로 변경하여 intercepting route 우회.

## Changes
- `packages/web/lib/hooks/useImageModalAnimation.ts:315` — `router.push` → `window.location.href`
- `packages/web/lib/hooks/useImageModalAnimation.ts:333` — dependency array에서 미사용 `router` 제거

## Verification
- TypeScript 컴파일 에러 없음 (`tsc --noEmit` 통과)
