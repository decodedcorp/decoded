# Chapter 4: 콘텐츠 생성 (Creation)

## 개요

이미지 업로드, AI 아이템 검출, 태그 편집 등 콘텐츠 생성과 관련된 모든 화면을 다룹니다.

## 화면 목록

| ID | 화면명 | 경로 | 상태 | 기능 ID |
|:---|:---|:---|:---:|:---|
| [SCR-CREA-01](./SCR-CREA-01-upload.md) | 업로드 | `/upload` | Draft | C-01 |
| [SCR-CREA-02](./SCR-CREA-02-detect.md) | AI 검출 결과 | `/upload/detect` | Draft | C-02 |
| [SCR-CREA-03](./SCR-CREA-03-edit.md) | 태그 편집 | `/upload/edit` | Draft | C-03, C-04 |

## 업로드 방법

| 방법 | 설명 | 지원 형식 |
|:---|:---|:---|
| 파일 선택 | 로컬 파일 선택 다이얼로그 | JPG, PNG, WebP |
| URL 입력 | 이미지 URL 직접 입력 | HTTP/HTTPS 이미지 |
| 드래그앤드롭 | 파일 드래그 후 드롭 영역에 놓기 | JPG, PNG, WebP |
| Instagram URL | Instagram 게시물 URL 파싱 | 공개 게시물만 |
