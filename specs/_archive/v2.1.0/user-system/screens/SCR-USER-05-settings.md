## 1. 화면 개요

- **목적**: 앱 언어 설정, 알림 설정, 계정 관리(로그아웃, 탈퇴)

## 3. UI 요소 정의

| UI ID | 구분 | 요소명 | 속성/상태 | 인터랙션/로직 |
|:---:|:---:|:---|:---|:---|
| **ICON-BACK** | 아이콘 | 뒤로가기 | - Icon: Arrow Left<br>- Size: 24×24px | **Click**: `/profile`로 이동 |
| **SEL-LANG** | 셀렉트 | 언어 선택 | - Options: 한국어, English<br>- Value: `ko` \| `en` | **Change**: <br>1. 전역 locale 상태 변경<br>2. UI 텍스트 즉시 적용<br>3. User Preference DB 업데이트 |
| **SEL-THEME** | 셀렉트 | 테마 선택 | - Options: Auto, Light, Dark<br>- Value: `auto` \| `light` \| `dark` | **Change**: <br>1. Theme Context 업데이트<br>2. CSS 변수 적용<br>3. localStorage 저장 |
| **TOGGLE-PUSH** | 토글 | 푸시 알림 | - On/Off 상태<br>- Default: On | **Change**: <br>1. 알림 권한 확인 (브라우저)<br>2. User Preference 업데이트 |
| **TOGGLE-EMAIL** | 토글 | 이메일 알림 | - On/Off 상태<br>- Default: On | **Change**: User Preference 업데이트 |
| **ROW-05** | 행 | 로그아웃 | - Style: Text Button<br>- Color: `text-primary` | **Click**: <br>1. `supabase.auth.signOut()` 호출<br>2. 로컬 스토리지 클리어<br>3. 홈 페이지로 리다이렉트 |
| **ROW-06** | 행 | 회원 탈퇴 | - Style: Text Button<br>- Color: `text-danger` | **Click**: <br>1. 탈퇴 확인 모달 표시<br>2. 확인 시 계정 삭제 API 호출<br>3. 로그인 페이지로 리다이렉트 |
| **ROW-07** | 행 | 이용약관 | - Clickable Row | **Click**: `/terms` (새 탭) |
| **ROW-08** | 행 | 개인정보 처리방침 | - Clickable Row | **Click**: `/privacy` (새 탭) |
| **ROW-09** | 행 | 버전 정보 | - Display Only | 앱 버전 표시 (package.json) |

## 5. 데이터 요구사항

### 5.1 API 호출

| API | Method | Endpoint | 호출 시점 | 응답 |
|:---|:---:|:---|:---|:---|
| 설정 조회 | GET | `/api/profile/settings` | 페이지 진입 시 | `{ language, theme, notifications }` |
| 설정 저장 | PATCH | `/api/profile/settings` | 설정 변경 시 | `{ success: boolean }` |
| 로그아웃 | - | `supabase.auth.signOut()` | 로그아웃 클릭 시 | - |
| 계정 삭제 | DELETE | `/api/profile/account` | 탈퇴 확인 시 | `{ success: boolean }` |

### 5.2 상태 관리

| 스토어 | 키 | 타입 | 설명 |
|:---|:---|:---|:---|
| Zustand (localeStore) | `locale` | `'ko' \| 'en'` | 현재 언어 설정 |
| Theme Context | `theme` | `'auto' \| 'light' \| 'dark'` | 현재 테마 |
| localStorage | `decoded_settings` | `UserSettings` | 게스트용 설정 저장 |

### 5.3 데이터 타입

```typescript
interface UserSettings {
  language: 'ko' | 'en';
  theme: 'auto' | 'light' | 'dark';
  notifications: {
    push: boolean;
    email: boolean;
  };
}
```

## 6. 에러 처리

| 에러 코드 | 상황 | 사용자 메시지 | 처리 방법 |
|:---:|:---|:---|:---|
| 401 | 인증 만료 | "세션이 만료되었습니다" | 로그인 페이지로 이동 |
| 403 | 권한 없음 | "권한이 없습니다" | 이전 페이지로 이동 |
| 500 | 저장 실패 | "설정 저장에 실패했습니다" | 재시도 안내 |

## 7. 접근성 (A11y)

- **키보드 네비게이션**: Tab으로 모든 컨트롤 접근
- **스크린 리더**: 토글에 `aria-label` 및 상태 표시
- **포커스 관리**: 모달 오픈 시 포커스 트랩, 닫을 때 원래 위치로
- **색상 대비**: 토글 On/Off 색상 + 아이콘으로 구분
