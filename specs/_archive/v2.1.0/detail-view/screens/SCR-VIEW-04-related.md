## 4. 상태 정의

## 5. 데이터 요구사항

### 5.1 API 호출

| API | Method | Endpoint | 호출 시점 | 응답 |
|:---|:---:|:---|:---|:---|
| 댓글 목록 | GET | `/api/images/:id/comments?page` | 페이지 로드 | `{ comments[], hasNext }` |
| 댓글 작성 | POST | `/api/images/:id/comments` | Submit | `{ comment }` |
| 댓글 수정 | PATCH | `/api/comments/:id` | Save | `{ comment }` |
| 댓글 삭제 | DELETE | `/api/comments/:id` | Delete 확인 | `{ success }` |
| 댓글 신고 | POST | `/api/comments/:id/report` | 신고 제출 | `{ success }` |

### 5.2 데이터 타입

```typescript
// 스마트 태그
interface SmartTagsProps {
  media?: {
    id: string;
    name: string;
    type: 'group' | 'show' | 'drama' | 'movie';
  };
  cast?: {
    id: string;
    name: string;
  }[];
  contextType?: ContextType;
}

// 컨텍스트 타입 아이콘 매핑
const contextIcons: Record<ContextType, string> = {
  stage: '🎤',
  airport: '✈️',
  mv: '🎬',
  photoshoot: '📸',
  drama: '🎭',
  variety: '📺',
  training: '🏃',
  casual: '👕',
};

// 댓글
interface Comment {
  id: string;
  userId: string;
  user: {
    displayName: string;
    avatarUrl?: string;
  };
  content: string;
  parentId?: string;
  replies?: Comment[];
  status: 'visible' | 'deleted' | 'hidden';
  createdAt: Date;
  updatedAt: Date;
}

// 댓글 훅
interface UseCommentsReturn {
  comments: Comment[];
  isLoading: boolean;
  hasNextPage: boolean;
  fetchNextPage: () => void;
  addComment: (content: string, parentId?: string) => Promise<void>;
  editComment: (id: string, content: string) => Promise<void>;
  deleteComment: (id: string) => Promise<void>;
  reportComment: (id: string, reason: string) => Promise<void>;
}
```

## 6. 에러 처리

| 에러 상황 | 사용자 메시지 | 처리 방법 |
|:---|:---|:---|
| 댓글 로드 실패 | "Failed to load comments" | 재시도 버튼 |
| 댓글 작성 실패 | "Failed to post comment" | 내용 유지 + 재시도 |
| 권한 없음 (수정/삭제) | "You can only edit your own comments" | Toast 알림 |
| 신고 중복 | "You already reported this comment" | Toast 알림 |
| 빈 댓글 | "Comment cannot be empty" | 제출 버튼 비활성화 |

## 7. 접근성 (A11y)

### 스마트 태그
- **키보드**: Tab으로 태그 간 이동, Enter로 네비게이션
- **ARIA**: `role="navigation"`, `aria-label="Content context"`

### 댓글
- **키보드**: Tab으로 요소 간 이동
- **스크린 리더**:
  - `aria-label="Comment by username, posted X time ago"`
  - `aria-live="polite"` (새 댓글 알림)
- **포커스**: 답글 작성 시 입력 폼으로 포커스 이동
